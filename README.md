## What is the purpose of this Template?

Laravel template with
- Laravel Sail for an easy development enviroment
- Laravel Octane for better performance
- PHPStan on level 10 for complete type safety
- Pint for linting

## Instructions
To start the project locally run:

``composer install``

``/vendor/bin/sail pnpm install``

``/vendor/bin/sail up``

``/vendor/bin/sail pnpm run dev``
